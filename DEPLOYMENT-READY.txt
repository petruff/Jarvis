QUICK START
===========

1. Setup Environment (1 min):
   cp .env.example .env
   # Edit .env with your API keys

2. Build & Start (2 min):
   docker-compose build
   docker-compose up -d
   sleep 30

3. Verify (1 min):
   docker-compose ps
   curl http://localhost:3000/health

4. Access Services:
   Frontend: http://localhost:5173
   Gateway: http://localhost:3001
   Monitoring: http://localhost:3100 (admin/admin)

===============

Database issues:
docker-compose exec jarvis-postgres pg_isready

Memory issues:
docker stats
docker-compose down -v
docker-compose up -d

## Prerequisites

Before starting, ensure you have:

- [ ] Node.js 18+ installed (`node --version`)
- [ ] npm 9+ installed (`npm --version`)
- [ ] Git installed (`git --version`)
- [ ] GitHub CLI (`gh`) installed and authenticated (`gh auth status`)
- [ ] An AI-powered IDE or Claude Code CLI

---

## Step 1: Install Electron AAOS Core

### Option A: npx Installation Wizard (Recommended)

```bash
# Run the interactive installation wizard
npx @electron-aaos/electron-aaos-core@latest

# Or create a new project with a specific name
npx @electron-aaos/electron-aaos-core@latest init my-project
cd my-project
```

### Option B: Clone Repository (Development)

```bash
git clone https://github.com/Electron AAOSAI/electron-aaos-core.git
cd electron-aaos-core
npm install
```

---

## Step 2: Verify Installation

Run the diagnostics command:

```bash
npx @electron-aaos/electron-aaos-core@latest doctor
```

Or if installed globally:

```bash
electron-aaos doctor
```

### Manual Verification

```bash
# Check core structure exists
ls -la .electron-aaos-core/

# Verify key directories
ls .electron-aaos-core/core/
ls .electron-aaos-core/development/agents/
```

Expected structure:

```
.electron-aaos-core/
├── core/               # Framework core (registry, health-check, orchestration)
├── development/        # Agents, tasks, workflows
├── product/            # Templates, checklists
└── infrastructure/     # Scripts, tools, integrations
```

---

## Step 3: Activate Your First Agent

Electron AAOS uses specialized agents for different tasks. In your AI-powered IDE or Claude Code CLI, type:

```
@electron-aaos-master
```

The agent will greet you and show available commands:

```
🎯 Electron AAOS Master ready!
Type *help to see available commands.
```

### Try These Commands

| Command   | Description                 |
| --------- | --------------------------- |
| `*help`   | Show all available commands |
| `*status` | Show project status         |
| `*agents` | List all available agents   |

## Step 5: Create Your First Story

Stories drive development in Electron AAOS. Activate the Product Owner and create one:

```
@po *create-story
```

Follow the prompts to define:

1. Story title
2. Description
3. Acceptance criteria
4. Priority

## Quick Reference

### Agent Commands

All agent commands use the `*` prefix:

```
*help          # Show help
*status        # Show status
*exit          # Exit current agent
```

### CLI Commands

```bash
# Installation and setup
npx @electron-aaos/electron-aaos-core@latest           # Run wizard
npx @electron-aaos/electron-aaos-core@latest doctor    # Run diagnostics
npx @electron-aaos/electron-aaos-core@latest info      # Show system info

# Development
npm run lint                           # Check code style
npm run typecheck                      # Check TypeScript types
npm test                               # Run unit tests
npm run validate:structure             # Validate Electron AAOS structure
```

## Troubleshooting

### "Command not found" errors

```bash
# Ensure Node.js is in PATH
node --version

# Clear npm cache if issues persist
npm cache clean --force
```

### Agent not responding

1. Ensure you're in an AI-powered IDE (Cursor, VS Code with Claude, etc.) or Claude Code CLI
2. Check you're using correct activation syntax: `@agent-name`
3. Verify agent file exists: `ls .electron-aaos-core/development/agents/`

### Permission errors

```bash
# Fix npm permissions (macOS/Linux)
sudo chown -R $(whoami) ~/.npm

# Or use a Node version manager (recommended)
# nvm, fnm, or volta
```

### Electron AAOS structure not found

```bash
# Reinstall Electron AAOS in current project
npx @electron-aaos/electron-aaos-core@latest install

# Or clone fresh
git clone https://github.com/Electron AAOSAI/electron-aaos-core.git
```

## Vision General

**@electron-aaos-master** (Orion) es el agente orquestador central del Electron AAOS, responsable por:

- Coordinar todos los agentes del sistema
- Gestionar workflows multi-agente
- Mantener el contexto global del proyecto
- Delegar tareas a agentes especializados

## Comandos Principales

| Comando | Descripcion |
|---------|-------------|
| `*help` | Mostrar comandos disponibles |
| `*status` | Status del sistema |
| `*delegate` | Delegar a agente especifico |
| `*workflow` | Ejecutar workflow |
